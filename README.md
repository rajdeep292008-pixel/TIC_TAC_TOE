# Cyber-Grid: Tic-Tac-Toe
A premium, futuristic Tic-Tac-Toe game built with a dark cyberpunk aesthetic. Features a smart Minimax-powered AI, glassmorphism UI, and procedural sound generation.

---

## 🎮 Game Features
- **Dual Modes:** Switch between classic "Player vs Player" or challenge the "Player vs AI" mode.
- **Intelligent AI:** Built using the **Minimax Algorithm** to ensure a challenging, unbeatable opponent.
- **Procedural Audio:** Features high-quality sound effects generated on-the-fly using the Web Audio API—no external files required.
- **Premium Design:** Glassmorphism UI, neon glow effects, and smooth 60 FPS animations.
- **Responsive:** Fully touch-friendly and optimized for mobile and desktop screens.

---

## 🚀 Technical Highlights
- **Architecture:** Modular JavaScript design for maintainable and clean code.
- **Graphics:** Hardware-accelerated CSS animations and custom neon gradients.
- **Performance:** Lightweight, single-file implementation with zero external libraries or dependencies.

---

## 🛠 How to Play
1. Open `cyber-tictactoe.html` in any modern web browser.
2. Select your game mode from the main menu.
3. If playing AI, use the "Minimax" difficulty logic to try and win (good luck!).
4. Use the "Reset" button to return to the main menu at any time.

---

## 💡 Why I Built This
This project was developed to explore the intersection of **Game Theory (AI)** and **Modern Web UI Design**. It demonstrates the ability to manage complex state logic while maintaining a high-end, arcade-style user experience.
